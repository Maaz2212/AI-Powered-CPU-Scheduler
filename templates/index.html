🐧 CFS — Completely Fair Scheduler

Era: Default Linux scheduler since kernel 2.6.23 (2007)

Core idea: Instead of predicting burst time, CFS tracks virtual runtime (vruntime) — how much CPU each process has received, weighted by its nice value. It always picks the process with the lowest vruntime, guaranteeing every process a fair share.

Formula: vruntime += Δt × (NICE_0_WEIGHT / process_weight)

vs Traditional: FCFS/SJF decide based on burst time → CFS decides based on vruntime (no prediction needed at all).

vs Our ML model: CFS optimises for fairness — our Random Forest optimises for burst time accuracy, enabling smarter SJF/SRTF scheduling decisions.
